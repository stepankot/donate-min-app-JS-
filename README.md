# React JS + FastAPI (Онлайн копилка с системой рейтинга)
## Мобильное веб-апп, приближенное к IOS style 
## С возможностью интеграции в telegram mini app
## Функционал:
* CRUD операции на вашу накопительную цель
* Система рейтинга пользователей в процентном соотношении
* Приятный Interface, сделанный по FIGMA макету
